# SEIR_Mendoza
COVID-19 SEIR + metapopulation model for Mendoza province in Argentina

The notebook can be run from the following link:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/ihem-institute/SEIR_Mendoza.git/master?filepath=COVID-19_SEIR_Mendoza.ipynb)

Or by locally cloning the repository and executing 'jupyter notebook' in the folder where the 'ipynb' resides. The notebook requires recent 'numpy', 'scipy' and 'matplotlib' packages to run.

Once the paper is published a link will be included here.
